## Introduction

> A simple blog system based on ThreeJS & Vue

Libraries I used:

* Vue.js
* Three.js
* OribitControls
* Shaderfrog-Runtime
* Stats.js
* dat.GUI
* GLSLEditor
* GLSLCanvas
* highlight.js
* highlight-linenumbers.js
* MathJax
* GSAP
* Rebound.js
* HTML5 device mockups
* MathJax
* Babel 
* Webpack
* ESLint
* Stylus
* vue-ghpages-blog
* vue-gh-pages

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for static page
npm run deploy
```
